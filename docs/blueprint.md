# **App Name**: Aura Interna

## Core Features:

- Authentication: User authentication via Firebase Authentication with email and password, including login, registration, and password recovery screens.
- Module Dashboard: Dashboard with five modules represented as cards or lists with icons/images. Names are customizable (Introdução, Estratégias, Implementação, Testes, Escala).
- Login Verification: Secure routing that redirects unauthenticated users to the login page. Session management managed using Firebase Authentication.
- Animated Navigation: Smooth page transitions to enhance the user experience. Transitions must be implemented using Nextjs.

## Style Guidelines:

- Primary color: Deep Liturgical Blue (#0B2E59) to symbolize faith and protection.
- Secondary color: Crimson Red (#8A1C21) to represent courage and devotion.
- Accent color: Glimmering Gold (#C7A348) to evoke royalty and divine presence.
- Background color: Pearl White (#FAF8F2) for soft backgrounds.
- Titles and sections: 'Cinzel Decorative' (serif) with Bold variations for main titles. Note: currently only Google Fonts are supported.
- Subtitles and buttons: 'Playfair Display' (serif), elegant with soft serifs. Note: currently only Google Fonts are supported.
- Body text, UI, AI Chat: 'Inter' (sans-serif) or 'Lato' (sans-serif), for high legibility on screens. Note: currently only Google Fonts are supported.
- Visually impactful design with a dark mode theme, soft gradients, and use of Tailwind CSS with Backdrop Blur, similar to Discord.
- Fully responsive design for both mobile and desktop.
